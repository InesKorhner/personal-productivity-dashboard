# Spacing System - Design Tokens

## Standardizovani Spacing Sistem

### Page Padding
- **Mobile**: `p-4` (16px)
- **Tablet+**: `md:p-6` (24px)
- **Desktop Grid**: `xl:p-6` (24px)

### Content Spacing
- **Main content vertical spacing**: `space-y-4` (16px)
- **Form elements**: `gap-4` (16px)
- **Section spacing**: `space-y-3` (12px)
- **Tight lists**: `space-y-1` (4px)

### Component Spacing
- **Button groups**: `gap-2` (8px)
- **Error message content**: `gap-3` (12px)
- **Error message buttons**: `gap-2` (8px)
- **Card padding**: `p-4` (16px)
- **Sidebar padding**: `p-4` (16px)

### Typography Spacing
- **Heading margin bottom**: `mb-4` (16px) za h1, `mb-3` (12px) za h2
- **Paragraph spacing**: `mb-2` (8px)
- **List item spacing**: `space-y-1` (4px)
